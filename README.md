# 💰 Micro-Budget App

A smart micro-budgeting dashboard that helps users track expenses, manage savings, and make better financial decisions.

--
## 🚀 Live Demo

https://microbuget-one.vercel.app/

---

## 📌 Features

* 🔐 Family Login System (Father, Mother, You)
* 💸 Expense Management (Add, Edit, Delete)
* 📊 Dashboard with Income, Expenses, Savings
* 🔮 Predictive Analytics for future spending
* 🎯 Savings Goals Tracking
* 🔍 Search & Category Filters
* ⚠️ Budget Overuse Alerts
* 🏆 Gamification Badges
* 📅 Daily Budget Indicator
* 👨‍👩‍👧 Family Expense Tracking (who added expense)

---

## ⚙️ Installation

```bash
git clone https://github.com/Varshitha-mandha/Micro-Buget.git
cd micro-budget-app
npm install
npm run dev
```
